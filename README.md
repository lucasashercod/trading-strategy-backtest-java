# Trading Strategy Backtest (Java)

Projeto de simulação de estratégias no mercado futuro (WIN/WDO).

## Funcionalidades
- Simulação de trades
- Estratégia baseada em máxima, mínima e preço justo
- Cálculo de lucro
- Métricas: lucro total, número de trades, win rate

## Objetivo
Validar estratégias operacionais com base em dados e regras objetivas.# trading-strategy-backtest-java
